# Sound Effects

This directory should contain the following sound files for the game:

- `click.mp3` - Button click sound (short, crisp)
- `reveal.mp3` - Word reveal sound (mysterious, magical)
- `win.mp3` - Victory sound (celebratory, upbeat)
- `lose.mp3` - Loss sound (dramatic, but not harsh)
- `success.mp3` - Success/confirmation sound (positive, satisfying)

## Recommended Sources

You can find free sound effects from:
- [Freesound.org](https://freesound.org) - Search for "button click", "reveal", "victory", etc.
- [Zapsplat](https://www.zapsplat.com) - Free sound effects library
- [Mixkit](https://mixkit.co/free-sound-effects/) - Free sound effects

## Format

- Format: MP3
- Recommended bitrate: 128-192 kbps
- Duration: 0.5-2 seconds (shorter for clicks, longer for win/lose)
- Volume: Normalized to avoid clipping

## Note

The game will work without sound files - it will simply continue silently if files are not found.
